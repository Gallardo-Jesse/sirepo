SRW (Synchrotron Radiation Workshop)

SRW (Synchrotron Radiation Workshop) computes synchrotron radiation from relativistic electrons in arbitrary magnetic fields and propagates the radiation wavefronts through optical beamlines.

The SRW source code is available on GitHub.

SRW is primarily supported by Oleg Chubar of NSLS-II at Brookhaven National Laboratory.

The Sirepo GUI is primarily supported by the development team at RadiaSoft LLC.
