Warp VND

Warp VND is a Sirepo-based app for high-fidelity modeling of thermionic converters, which are a type of vacuum nanoelectronic device (VND).

Thermionic converters generate electricity directly from heat, via thermionic emission of electrons.

The underlying physics kernel is based on Warp, a massively parallel PIC code.
