RsLinac

RsLinac simulates traveling wave (TW) electron linacs with beam loading and beam transport with a variety of standard elements.

RsLinac source code is open source on GitHub.

The underlying physics code, called ‘Hellweg’, was initially developed in the RF laboratory of National Research Nuclear University (MEPhl). The code is primarily developed by Sergey Kutsaev at RadiaBeam Technologies, with support from RadiaSoft personnel.
