OPAL

OPAL is a framework for charged-particle optics in accelerator structures and beam lines. The code is derived from MAD9P and is based on the CLASSIC class library. IPPL (Independent Parallel Particle Layer) is the framework which provides parallel particles and elds using a data parallel ansatz. OPAL runs on your laptop as well as on the largest HPC clusters available today. There are two executables built from the OPAL framework:

  • OPAL-CYCL tracks hadrons through cyclotrons with 3D space charge, including neighboring turns.
  • OPAL-T is used to model electron guns, injectors and complete linacs.

OPAL is supported by the AMAS group at PSI.
