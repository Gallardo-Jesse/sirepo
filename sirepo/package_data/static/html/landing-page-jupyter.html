JupyterHub Server

RadiaSoft provides community physics code on a public HPC server --

   Particle accelerators: elegant, Warp, Synergia and OPAL

   X-ray Optics: SRW and Shadow3

  • All codes are pre-installed.
  • Users are expected to have some familiarity with JupyterHub servers,
  • Experience using a selected code is required.
  • A short video on using JupyterHub is provided.
