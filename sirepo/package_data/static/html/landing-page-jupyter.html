Jupyter Server

RadiaSoft provides access to a full suite of community applications for simulation and design:

  • Particle accelerators: elegant, Warp, Syngergia, OPAL
  • X-Ray Optics: SRW, Shadow3
  • Free electron lasers: Genesis
  • All codes are pre-installed.
  • Users are expected to have some experience with Jupyter servers and must have experience with the selected application.
  • All codes can run in your browser from a LINUX terminal.
  • Some codes are also supported for use from Jupyter notebooks (see chart)
  • The codes run on parallel Linux servers provided by RadiaSoft.
