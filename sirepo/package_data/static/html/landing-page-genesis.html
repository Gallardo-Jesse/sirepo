Genesis

Genesis is a time-dependent 3D free electron laser (FEL) code, designed primarily for the simulation of single-pass FELs, both in amplifier and SASE configurations. The flexible input can be used to address FEL oscillators and multi-stage configurations.

The code was written by Sven Reiche as a part of the Ph.D. thesis at DESY, Germany, and then further extended at UCLA and DESY.
